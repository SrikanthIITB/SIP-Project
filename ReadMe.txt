System Requirement - Windows 
C to be installed earlier

---Use of Application 
1) Open folder conating file name SIP_Project_2021
2) Double click , Open file name SIP_Project.exe
3) A new window will be Open
4) Right click on "Upload Target Image" for --> input image
5) Right click on "Upload Reference Image" for --> desired image histogram
6) click on Go button on left corner
7) New enhanced image will be display with below histogram
8) Click on right and left histogram of it projection
9) save image with left click for any histogram and enchanced image.
10) Right up corner click on options to "Refresh all" for fresh set of image
11) Click on "Exit" to close the panel.
